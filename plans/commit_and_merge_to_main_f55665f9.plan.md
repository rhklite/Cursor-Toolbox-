# Commit and Merge to Main

## Current state

- Branch: `feature/experiment-tracker` (11 commits ahead of `main`)
- Uncommitted local changes from this session:
  - App icon (`AppIcon.icns`) + `Info.plist` CFBundleIconFile
  - `build_app.sh` icon-conflict fix (removes `applet.icns`, `Assets.car`, `CFBundleIconName`)
  - `menubar_app.py` unified lifecycle (`--docker` mode, PID guard, `docker compose down` on quit)
  - `launch` script (`exec` into menubar app, duplicate-launch guard)
  - `docker-compose.yml` restart policy change (`unless-stopped` -> `no`)
- `build/` contains machine-specific osacompile artifacts (binaries, code signatures) that should not be tracked

## Steps

### 1. Add `build/` and `dist/` to [.gitignore](.gitignore) and remove from tracking

The `build/` directory contains osacompile output (binaries, `.scpt`, code signatures, `.car`) that are machine-specific and regenerated by `build_app.sh`. Stop tracking them.

```
build/
dist/
```

Then `git rm -r --cached build/` to untrack without deleting.

### 2. Commit local changes

Two commits to keep the history clean:

1. `**<chore> Add build/ and dist/ to gitignore**` -- `.gitignore` update + `git rm --cached`
2. `**<feature> Add app icon, unified lifecycle, and Docker mode for menubar**` -- all remaining changes:
  - `AppIcon.icns` + `Info.plist`
  - `build_app.sh` icon-conflict fix
  - `menubar_app.py` Docker mode + PID guard
  - `launch` script lifecycle handoff
  - `docker-compose.yml` restart policy

### 3. Merge into `main`

```bash
git checkout main
git merge feature/experiment-tracker
```

This is a fast-forward-eligible merge (main has no divergent commits). If preferred, `--no-ff` creates an explicit merge commit.

### 4. Push both branches to remote

```bash
git push origin main
git push origin feature/experiment-tracker
```

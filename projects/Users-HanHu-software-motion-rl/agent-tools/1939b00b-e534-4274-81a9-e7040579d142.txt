+# iron_turbo (distributed primitives for RL locomotion)
+
+`iron_turbo` is a small drop-in module that adds **RL-specific distributed helpers** on top of
+PyTorch distributed + HuggingFace Accelerate.
+
+Scope of this README: **distributed training only** (no RL algorithm content). Algorithmic contents like CTS, STS, gSDE, distributional critic, etc. will be included in latter release.
+
+Ultimately, `iron_turbo` aims to be an “Accelerate for locomotion RL”: a small set of abstractions that make multi-GPU training easy to adopt in existing codebases. On top of that, it standardizes best-practice integrations (engineering and algorithmic) and ensures they work seamlessly under distributed execution, improving both throughput and final policy quality.
+
+---
+
+## Requirements
+
+Add:
+
+```bash
+pip install accelerate
+```
+to `fuyao_train.sh`. And drop iron_turbo into `humanoid-gym/humanoid/algo/`.
+
+
+---
+
+## What you get
+
+### Core primitives (model/grad + RL stats)
+
+* `is_distributed()`: true iff `torch.distributed` is initialized and `world_size > 1`
+* `get_world_size()`: robust world size (returns 1 if not distributed)
+* `broadcast_parameters(*modules)`: broadcast params from rank 0 to all ranks (init sync)
+* `sync_gradients(*modules)`: manual all-reduce grads (flatten → all_reduce → unfold)
+* `standardize_tensor(tensor, accelerator)`: global standardization (mean/std over all ranks)
+* `reduce_mean(tensor, accelerator)`: global mean helper
+
+### RL-specific wrapper modules (drop-in)
+
+* `SyncNormalizerTorch(base_normalizer)`
+
+  * Wrap a local `NormalizerTorch` and **auto-sync its `update()`** when distributed
+  * Saves/loads via `state_dict()` transparently (the base is a registered submodule)
+
+* `SyncRunningMeanStd(base_rms)`
+
+  * Same idea for a local `RunningMeanStdTorch`-style module (sync on `update()`)
+
+### Logging / data-structure helpers
+
+* `sync_tensor(tensor, accelerator, reduction="mean"|"sum")`
+* `sync_tensor_mean(tensor, accelerator)`: local mean → global mean
+* `sync_tensor_dict(dict, accelerator, device)`: sync dict values for logging

+## Migration guide: single-GPU → distributed (DDP-style)
+
+### Step 0 - A DDP mental model (RL-oriented)
+
+* Each rank runs its own envs and collects rollouts locally.
+* During learning:
+
+  * `broadcast_parameters(...)` ensures identical initialization across ranks.
+  * `sync_gradients(...)` makes updates equivalent to data-parallel training.
+  * RL-specific stats (normalizers / advantage standardization / logs) must be reduced globally.
+
+That’s it: keep rollout local, synchronize what must be global.
+
+Below shows the *minimum* changes needed in typical locomotion PPO/AMP code.
+
+### Step 1 — Enable distributed runtime in `train.py` (Accelerator + device/rank/seed + launch)
+
+This is the **only `train.py` change** you need before touching PPO/AMP code: create the distributed context, derive device/rank, set a rank-safe seed, and gate logging/saving on the main process.
+
+**Runnable `train.py` skeleton (copy-paste):**
+
+```python
+# train.py
+import argparse
+from accelerate import Accelerator
+from iron_turbo.distributed_utils import get_distributed_seed
+
+def parse_args():
+    p = argparse.ArgumentParser()
+    p.add_argument("--distributed", action="store_true")
+    p.add_argument("--seed", type=int, default=1)
+    return p.parse_args()
+
+def main():
+    args = parse_args()
+
+    # Distributed context (only when enabled)
+    accelerator = Accelerator() if args.distributed else None
+
+    # Device / rank (single source of truth)
+    device = accelerator.device if accelerator else "cuda:0"
+    local_rank = accelerator.local_process_index if accelerator else 0
+
+    # Rank-safe seed
+    seed = get_distributed_seed(args.seed, accelerator)
+
+    # Main-process guard
+    is_main = accelerator.is_main_process if accelerator else True
+    if is_main:
+        print(f"[main] device={device}, local_rank={local_rank}, seed={seed}")
+
+    # ---- your existing setup starts here ----
+    # build envs on `device`, build models on `device`, run training loop, etc.
+
+if __name__ == "__main__":
+    main()
+```
+
+**Launch (local torchrun):**
+
+```bash
+torchrun --standalone --nproc_per_node=4 train.py --distributed
+```

+"""
+Core RL-specific distributed utilities for iron_turbo.
+
+This module provides:
+    - is_distributed: Check if running in distributed mode.
+    - broadcast_parameters: Broadcast model parameters from rank 0 to all ranks.
+    - sync_gradients: Synchronize gradients across all ranks.
+    - SyncRunningMeanStd: A wrapper that adds distributed synchronization to a local
+      RunningMeanStdTorch instance without modifying the original class.
+    - standardize_tensor: A functional normalizer for global advantage normalization.
+    - reduce_mean: Helper for computing global means across processes.
+
+These components use Accelerate's `reduce` for backend-agnostic distributed operations,
+or torch.distributed for low-level collective operations.
+"""
+
+from __future__ import annotations
+
+from typing import TYPE_CHECKING, Optional

+import torch
+import torch.distributed as dist

+def is_main_process(accelerator: Optional["Accelerator"] = None) -> bool:
+    """Check if this is the main process (rank 0).
+
+    Args:
+        accelerator: Optional Hugging Face Accelerate Accelerator instance.
+            If provided, uses `accelerator.is_main_process`. If None, falls back
+            to torch.distributed rank checks (rank 0) when available/initialized,
+            otherwise assumes single-process execution and returns True.
+
+    Returns:
+        True if this process should perform main-process-only actions (logging,
+        checkpointing, spawning subprocesses, etc.).
+    """
+    if accelerator is not None:
+        return accelerator.is_main_process
+
+    # Fallback for codepaths that don't use Accelerate:
+    # - If torch.distributed is initialized: main process == rank 0
+    # - Otherwise: single process => main
+    try:
+        if dist.is_available() and dist.is_initialized():
+            return dist.get_rank() == 0
+    except Exception:
+        # Defensive: treat unknown dist state as single-process.
+        pass
+
+    return True
+
+
+def get_world_size(default: int = 1) -> int:
+    """Robustly get torch.distributed world size.
+
+    Works for:
+      - normal single-process training (no dist init)
+      - DDP/torchrun/torch.distributed.run (dist initialized)
+      - environments where torch.distributed isn't available/built
+
+    Args:
+        default: Value to return when distributed is not available/initialized.
+
+    Returns:
+        World size (>= 1).
+    """
+    try:
+        if dist.is_available() and dist.is_initialized():
+            ws = dist.get_world_size()
+            return int(ws) if ws and ws > 0 else default
+    except Exception:
+        # Covers cases like partially-initialized process groups or backend issues.
+        pass
+    return default
+

+def is_distributed() -> bool:
+    """Check if running in distributed mode.
+
+    Returns:
+        True if torch.distributed is initialized and world_size > 1.
+    """
+    return get_world_size() > 1
+

+def broadcast_parameters(*modules: nn.Module, src: int = 0) -> None:
+    """Broadcast parameters of multiple modules from src rank to all ranks.
+
+    No-op if not in distributed mode.
+
+    Args:
+        *modules: One or more nn.Module instances whose parameters to broadcast.
+        src: Source rank to broadcast from (default 0).
+
+    Example:
+        >>> broadcast_parameters(actor_critic, discriminator)
+    """
+    if not is_distributed():
+        return
+
+    for module in modules:
+        for param in module.parameters():
+            dist.broadcast(param.data, src=src)
+
+
+def sync_gradients(*modules: nn.Module) -> None:
+    """Synchronize gradients of multiple modules across all ranks.
+
+    Uses flatten-reduce-unfold pattern for efficiency (single collective op
+    per module instead of one per parameter).
+
+    No-op if not in distributed mode.
+
+    Note: If using Accelerate, prefer `accelerator.backward()` which handles
+    gradient sync automatically. This function is for manual gradient sync
+    in non-Accelerate setups.
+
+    Args:
+        *modules: One or more nn.Module instances whose gradients to synchronize.
+
+    Example:
+        >>> loss.backward()
+        >>> sync_gradients(actor_critic, discriminator)
+        >>> optimizer.step()
+    """
+    if not is_distributed():
+        return
+
+    world_size = dist.get_world_size()
+
+    for module in modules:
+        # Flatten all gradients into one tensor
+        grads = [p.grad.reshape(-1) for p in module.parameters() if p.grad is not None]
+        if not grads:
+            continue
+
+        all_grads = torch.cat(grads)
+
+        # All-reduce with average
+        dist.all_reduce(all_grads, op=dist.ReduceOp.SUM)
+        all_grads /= world_size
+
+        # Unfold back to each parameter
+        offset = 0
+        for param in module.parameters():
+            if param.grad is not None:
+                numel = param.numel()
+                param.grad.data.copy_(all_grads[offset : offset + numel].view_as(param.grad))
+                offset += numel
+
+

+# Accelerate-based distributed utilities
+# =============================================================================

+def reduce_mean(tensor: torch.Tensor, accelerator: Optional["Accelerator"] = None) -> torch.Tensor:
+    """Compute the global mean of a tensor across all processes.
+
+    Args:
+        tensor: Input tensor to compute global mean for.
+        accelerator: Accelerator instance. If None, returns local mean.
+
+    Returns:
+        Scalar tensor containing the global mean.
+    """
+    if accelerator is None or accelerator.num_processes == 1:
+        return tensor.mean()
+
+    # Sum reduction across all processes
+    local_sum = tensor.sum()
+    local_count = torch.tensor(tensor.numel(), dtype=tensor.dtype, device=tensor.device)
+
+    # Stack for a single reduce call
+    stats = torch.stack([local_sum, local_count])
+    global_stats = accelerator.reduce(stats, reduction="sum")
+
+    global_sum, global_count = global_stats[0], global_stats[1]
+    return global_sum / global_count
+
+
+def standardize_tensor(
+    tensor: torch.Tensor,
+    accelerator: Optional["Accelerator"] = None,
+    eps: float = 1e-8,
+) -> torch.Tensor:
+    """Standardize a tensor to zero mean and unit variance.
+
+    If an Accelerator is provided and has multiple processes, computes global
+    statistics across all ranks using Accelerate's reduce operation.
+
+    This replaces the manual distributed advantage normalization logic in
+    RolloutStorage.compute_returns.
+
+    Args:
+        tensor: The tensor to standardize (e.g., advantages).
+        accelerator: Optional Accelerator instance for distributed training.
+            If None or single-process, performs local standardization.
+        eps: Small constant for numerical stability.
+
+    Returns:
+        Standardized tensor with approximately zero mean and unit variance.
+
+    Example:
+        >>> # In RolloutStorage.compute_returns:
+        >>> self.advantages = standardize_tensor(self.advantages, accelerator)
+    """
+    if accelerator is None or accelerator.num_processes == 1:
+        # Local normalization (single GPU)
+        return (tensor - tensor.mean()) / (tensor.std() + eps)
+
+    # Distributed: compute global mean and std using reduce
+    local_sum = tensor.sum()
+    local_count = torch.tensor(tensor.numel(), dtype=tensor.dtype, device=tensor.device)
+
+    # All-reduce to get global sum and count
+    stats = torch.stack([local_sum, local_count])
+    global_stats = accelerator.reduce(stats, reduction="sum")
+    global_sum, global_count = global_stats[0], global_stats[1]
+
+    global_mean = global_sum / global_count
+
+    # Compute global variance using Welford's parallel algorithm
+    local_sq_diff_sum = ((tensor - global_mean) ** 2).sum()
+    global_sq_diff_sum = accelerator.reduce(local_sq_diff_sum, reduction="sum")
+    global_var = global_sq_diff_sum / global_count
+    global_std = torch.sqrt(global_var + eps)
+
+    return (tensor - global_mean) / global_std
+

+class SyncRunningMeanStd(nn.Module):
+    """A transparent wrapper that adds distributed synchronization to a local RunningMeanStdTorch.
+
+    This follows the Minimum Invasion Principle: the underlying RunningMeanStdTorch
+    remains a simple, local-only implementation. Distributed behavior is added via
+    this wrapper layer.
+
+    The wrapper auto-detects distributed mode using is_distributed() and:
+    - In distributed mode: intercepts `update()` calls, computes local statistics,
+      synchronizes them globally using torch.distributed.all_reduce, then updates
+      the underlying instance with the global values.
+    - In single-process mode: acts as a transparent pass-through to the base instance.
+
+    Save/load works transparently via state_dict() since base is registered as a submodule.
+
+    Args:
+        base: The local RunningMeanStdTorch instance to wrap.
+
+    Example:
+        >>> from humanoid.algo.amp_ppo.utils import RunningMeanStdTorch
+        >>> base_normalizer = RunningMeanStdTorch(device, shape=(obs_dim,))
+        >>> sync_normalizer = SyncRunningMeanStd(base_normalizer)  # no accelerator needed!
+        >>> sync_normalizer.update(observations)  # Auto-syncs if distributed
+        >>>
+        >>> # Saving - just save directly
+        >>> torch.save({'normalizer': sync_normalizer.state_dict()}, 'checkpoint.pt')
+        >>> # Loading - just load directly
+        >>> sync_normalizer.load_state_dict(loaded_dict['normalizer'])
+    """
+
+    def __init__(self, base: nn.Module):
+        super().__init__()
+        self.base = base
+
+    @property
+    def mean(self) -> torch.Tensor:
+        """Access the running mean from the base instance."""
+        return self.base.mean
+
+    @property
+    def var(self) -> torch.Tensor:
+        """Access the running variance from the base instance."""
+        return self.base.var
+
+    @property
+    def count(self) -> torch.Tensor:
+        """Access the sample count from the base instance."""
+        return self.base.count
+
+    def forward(self, x: torch.Tensor) -> torch.Tensor:
+        """Forward pass delegates to base if it has a forward method."""
+        return self.base(x) if hasattr(self.base, 'forward') else x
+
+    @torch.jit.unused
+    def update(self, arr: torch.Tensor) -> None:
+        """Update running statistics with global synchronization if distributed.
+
+        In distributed mode: computes local batch statistics, synchronizes them
+        across all processes using torch.distributed.all_reduce, then updates
+        the base instance with global values.
+
+        In single-process mode: delegates directly to base.update().
+
+        Args:
+            arr: Tensor of shape (batch_size, *feature_dims) to update statistics with.
+        """
+        if not is_distributed():
+            # Pass-through to local update
+            self.base.update(arr)
+            return
+
+        # Compute local statistics with shifted algorithm for numerical stability
+        local_batch_size = arr.shape[0]
+        world_size = dist.get_world_size()
+        global_batch_size = world_size * local_batch_size
+
+        # Shift by current mean for numerical stability
+        x_shifted = arr - self.base.mean
+        local_sum_shifted = torch.sum(x_shifted, dim=0, keepdim=True)
+        local_sum_sq_shifted = torch.sum(x_shifted.pow(2), dim=0, keepdim=True)
+
+        # Sync stats across all processes (single all_reduce call for efficiency)
+        stats_to_sync = torch.cat([local_sum_shifted, local_sum_sq_shifted], dim=0)
+        dist.all_reduce(stats_to_sync, op=dist.ReduceOp.SUM)
+        global_sum_shifted, global_sum_sq_shifted = stats_to_sync[0], stats_to_sync[1]
+
+        # Compute global batch mean and variance
+        batch_mean_shifted = global_sum_shifted / global_batch_size
+        batch_var = global_sum_sq_shifted / global_batch_size - batch_mean_shifted.pow(2)
+        batch_mean = batch_mean_shifted + self.base.mean
+
+        # Update the base instance with global statistics
+        self.base.update_from_moments(batch_mean, batch_var, global_batch_size)
+
+
+class SyncNormalizerTorch(nn.Module):
+    """Drop-in NormalizerTorch wrapper: auto-sync update() iff distributed."""
+    def __init__(self, base: nn.Module):
+        super().__init__()
+        self.base = base  # registered submodule => saved in state_dict
+
+    def __getattr__(self, name):
+        # delegate everything unknown to base (but avoid recursion)
+        if name in {"base", "_modules", "_parameters", "_buffers"}:
+            return super().__getattr__(name)
+        try:
+            return super().__getattr__(name)
+        except AttributeError:
+            return getattr(self.base, name)
+
+    @torch.no_grad()
+    def update(self, arr: torch.Tensor) -> None:
+        if not is_distributed():
+            return self.base.update(arr)
+
+        # compute local moments relative to current mean for stability
+        local_n = arr.shape[0]
+        world = dist.get_world_size()
+        global_n = local_n * world
+
+        x = arr - self.base.mean
+        local_sum = x.sum(dim=0, keepdim=True)
+        local_sumsq = (x * x).sum(dim=0, keepdim=True)
+
+        stats = torch.cat([local_sum, local_sumsq], dim=0)
+        dist.all_reduce(stats, op=dist.ReduceOp.SUM)
+        gsum, gsumsq = stats[0], stats[1]
+
+        mean_shift = gsum / global_n
+        var = gsumsq / global_n - mean_shift.pow(2)
+        mean = mean_shift + self.base.mean
+
+        self.base.update_from_moments(mean, var, global_n)

+    - sync_tensor: Synchronize a tensor across ranks (output shape == input shape)
+    - sync_tensor_mean: Compute local mean, then sync that scalar across ranks
+    - sync_tensor_dict: Synchronize a dict of tensors across ranks
+    - get_distributed_seed: Generate per-rank deterministic seeds
+"""

+def sync_tensor(
+    tensor: torch.Tensor,
+    accelerator: Optional["Accelerator"] = None,
+    reduction: str = "mean",
+) -> torch.Tensor:
+    """Synchronize a tensor across all processes.
+
+    Args:
+        tensor: Tensor to synchronize (any shape).
+        accelerator: Accelerator instance. If None, returns tensor unchanged.
+        reduction: Reduction operation - "mean" or "sum".
+
+    Returns:
+        Synchronized tensor with the same shape as input.
+    """
+    if accelerator is None or accelerator.num_processes == 1:
+        return tensor
+
+    return accelerator.reduce(tensor, reduction=reduction)
+
+
+def sync_tensor_mean(
+    tensor: torch.Tensor,
+    accelerator: Optional["Accelerator"] = None,
+) -> torch.Tensor:
+    """Compute local mean of a tensor, then synchronize across all processes.
+
+    This is useful when each rank has a batch of values and you want the
+    global mean across all batches.
+
+    Args:
+        tensor: Tensor to compute mean and synchronize.
+        accelerator: Accelerator instance. If None, returns local mean.
+
+    Returns:
+        Scalar tensor containing the synchronized mean.
+
+    Example:
+        >>> # Sync AMP reward mean across ranks
+        >>> amp_reward_mean = sync_tensor_mean(amp_reward, accelerator)
+        >>> # Sync KL divergence mean for adaptive LR
+        >>> kl_mean = sync_tensor_mean(kl, accelerator)
+    """
+    local_mean = tensor.mean() if tensor.dim() > 0 else tensor
+
+    if accelerator is None or accelerator.num_processes == 1:
+        return local_mean
+
+    return accelerator.reduce(local_mean, reduction="mean")

+def sync_tensor_dict(
+    tensor_dict: Dict[str, torch.Tensor],
+    accelerator: Optional["Accelerator"] = None,
+    device: Optional[torch.device] = None,
+) -> Dict[str, float]:
+    """Synchronize a dictionary of tensors across all processes.
+
+    Each value is synchronized using mean reduction, then converted to a
+    Python float for logging.
+
+    Args:
+        tensor_dict: Dictionary of name -> tensor pairs.
+        accelerator: Accelerator instance. If None, returns dict with .item() values.
+        device: Device to create tensors on (used if values need conversion).
+
+    Returns:
+        Dictionary with synchronized values as Python floats.
+    """
+    if accelerator is None or accelerator.num_processes == 1:
+        return {k: (v.item() if isinstance(v, torch.Tensor) else v) for k, v in tensor_dict.items()}
+
+    synced_dict = {}
+    # Use deterministic key order for identical collective op order on all ranks
+    for key in sorted(tensor_dict.keys()):
+        value = tensor_dict[key]
+        if not isinstance(value, torch.Tensor):
+            value = torch.as_tensor(value, device=device, dtype=torch.float32)
+        synced_value = accelerator.reduce(value, reduction="mean")
+        synced_dict[key] = synced_value.item()
+
+    return synced_dict

+def get_distributed_seed(base_seed: int, accelerator: Optional["Accelerator"] = None) -> int:
+    """Get a unique seed for this process based on rank.
+
+    Args:
+        base_seed: Base seed value.
+        accelerator: Accelerator instance for getting process index.

+    Returns:
+        Seed value unique to this process (base_seed + rank).
+    """
+    if accelerator is None:
+        return base_seed
+    return base_seed + accelerator.process_index

+_DTYPE_TO_CODE = {
+    torch.float32: 0,
+    torch.float16: 1,
+    torch.bfloat16: 2,
+    torch.float64: 3,
+    torch.int64: 4,

+def _to_device_tensor(x: Union[np.ndarray, torch.Tensor], device: torch.device) -> torch.Tensor:
+    """Convert numpy array or tensor to a torch tensor on the specified device."""
+    if isinstance(x, np.ndarray):
+        return torch.from_numpy(x).to(device)

+@torch.no_grad()
+def broadcast_overwrite_from_src(
+    xs: Sequence[Union[np.ndarray, torch.Tensor]],
+    *,
+    src: int = 0,
+    device: Union[str, torch.device] = "cuda",
+    return_numpy: bool = False
+) -> Tuple[torch.Tensor, ...]:
+    """
+    Generic meta-broadcast that supports arbitrary shapes/dtypes.
+
+    Usage:
+        xs_out = broadcast_overwrite_from_src([a, b, c], src=0, device="cuda")
+        a, b, c = xs_out
+
+    Semantics:
+      - Must be called on ALL ranks when distributed.
+      - Only `src` rank's tensors/arrays are authoritative.
+      - Other ranks' inputs are ignored and may have different shapes/dtypes.
+      - Broadcasts metadata (dtype_code, ndim, shape) then payload for each item.
+      - Returns torch tensors on `device` (CUDA if device is cuda).
+
+    Preconditions:
+      - torch.distributed is initialized and world_size > 1 (guard with is_distributed()).
+      - `xs` length must be identical on all ranks (i.e., same number of items).
+      - Each item must be convertible to a torch tensor on src rank.
+    """
+    if not is_distributed():
+        raise RuntimeError("broadcast_overwrite_from_src() is distributed-only; guard it with is_distributed().")
+
+    dev = torch.device(device)
+    r = dist.get_rank()
+
+    # Ensure all ranks have same number of items.
+    # (We avoid Python-object broadcast for this; just assume caller passes same-length xs.)
+    k = len(xs)
+
+    out: List[torch.Tensor] = []
+
+    for i in range(k):
+        if r == src:
+            t_src = _to_device_tensor(xs[i], dev).contiguous()
+            code = _DTYPE_TO_CODE.get(t_src.dtype, None)
+            if code is None:
+                raise ValueError(f"Unsupported dtype for broadcast: {t_src.dtype}")
+            ndim = t_src.ndim
+        else:
+            t_src = None
+            code = 0
+            ndim = 0
+
+        # ---- broadcast (dtype_code, ndim) as CUDA int64 tensor ----
+        if r == src:
+            meta = torch.tensor([code, ndim], dtype=torch.int64, device=dev)
+        else:
+            meta = torch.empty(2, dtype=torch.int64, device=dev)
+        dist.broadcast(meta, src=src)
+
+        dtype = _CODE_TO_DTYPE[int(meta[0].item())]
+        ndim = int(meta[1].item())
+
+        # ---- broadcast shape (ndim int64) ----
+        if r == src:
+            shape_t = torch.tensor(list(t_src.shape), dtype=torch.int64, device=dev)  # type: ignore[union-attr]
+        else:
+            shape_t = torch.empty(ndim, dtype=torch.int64, device=dev)
+        dist.broadcast(shape_t, src=src)
+        shape = [int(v) for v in shape_t.tolist()]
+
+        # ---- allocate / coerce on src, allocate on others ----
+        if r != src:
+            t = torch.empty(shape, dtype=dtype, device=dev)
+        else:
+            t = t_src  # type: ignore[assignment]
+            if t.dtype != dtype:
+                t = t.to(dtype)
+            t = t.contiguous()
+
+        # ---- broadcast payload ----
+        dist.broadcast(t, src=src)
+        out.append(t.cpu().numpy() if return_numpy else t)
+
+    return tuple(out)

+def tensor_fingerprint_i64(t: torch.Tensor, stride: int = 1024) -> torch.Tensor:
+    """
+    Deterministic int64 fingerprint of tensor bits (subsampled).
+    Returns a 1-element int64 CUDA tensor.
+    """
+    x = t.contiguous().view(-1)
+
+    # reinterpret raw bits -> signed integer stream
+    if x.dtype == torch.float32:
+        bits = x.view(torch.int32).to(torch.int64)
+    elif x.dtype in (torch.float16, torch.bfloat16):
+        bits = x.view(torch.int16).to(torch.int64)
+    elif x.dtype == torch.float64:
+        bits = x.view(torch.int64)
+    elif x.dtype in (torch.int64, torch.int32, torch.int16, torch.int8, torch.uint8, torch.bool):
+        bits = x.to(torch.int64)
+    else:
+        raise TypeError(f"unsupported dtype: {x.dtype}")
+
+    bits = bits[::stride]
+    const = 6364136223846793005  # odd constant; overflow is fine/deterministic
+    fp = (bits * const).sum(dtype=torch.int64)
+    return fp

+@torch.no_grad()
+def assert_same_across_ranks(t: torch.Tensor, name: str = "tensor"):
+    fp = tensor_fingerprint_i64(t)
+    world = dist.get_world_size()
+    gathered = [torch.empty_like(fp) for _ in range(world)]
+    dist.all_gather(gathered, fp)
+    if dist.get_rank() == 0:
+        vals = [int(x.item()) for x in gathered]
+        if len(set(vals)) != 1:
+            raise RuntimeError(f"{name} differs across ranks. fingerprints={vals}")

+# Migration Guide - 如何使用 IronTurbo 将 motion_rl 改造成并行版本
+
+> **目标**：以最小改动将 motion_rl 改造成支持单机多卡的 DDP-style 分布式 RL 训练。核心思路如下：
+>
+> 1. 每张 GPU 独立运行环境进行并行采样；
+> 2. 初始网络参数保持一致，梯度正确同步；
+> 3. 关键统计量与归一化在全局范围内保持一致；
+> 4. 仅在主进程进行日志记录和模型保存。
+>
+> 本文的迁移路径与当前代码结构一致：**不将模型封装进 DDP**，而是采用 **DDP-style RL（手动同步梯度）** 的方式。
+
+## 核心概念释义
+
+在开始迁移步骤前，了解以下分布式训练中的基本概念有助于理解后续操作：
+
+*   **Rank**: 进程中一个独立的编号，用于在分布式组中唯一标识一个进程。
+*   **Global Rank / Rank**: 在整个分布式组（可能跨多台机器）中的唯一进程ID。通常 `rank=0` 的进程被视为主进程。
+*   **Local Rank**: 在**单台机器内部**的进程编号。例如，在一台有4张GPU的机器上，会有4个进程，其 `local_rank` 分别为 0, 1, 2, 3。它通常用于绑定进程到对应的物理GPU (`cuda:{local_rank}`)。
+*   **World Size**: 分布式组中进程的总数（即总的GPU数量）。
+*   **DDP-style RL (手动同步)**: 一种分布式策略，其中模型**不**被包装为 `torch.nn.parallel.DistributedDataParallel` 模块。我们手动管理不同进程间模型参数的初始一致性、梯度的同步以及统计量的聚合。这种方式提供了更大的灵活性（例如，可以方便地调用模型的自定义方法），但需要显式处理同步逻辑。
+
+## Step 0：启动方式（单机多卡）
+
+**内部启动机制：**
+在 `fuyao_train.sh` 中，使用 `torch.distributed.run` 来启动分布式训练。具体启动命令如下：

+```bash

+```

+**关键参数说明：**
+- `--nnodes=1`：指定节点数为 1（单机多卡）
+- `--nproc_per_node=4`：指定每个节点使用 4 个进程（即 4 张 GPU 卡）

+## Step 4：确保各 rank 的模型参数完全一致（初始化或加载后广播）
+
+在 DDP-style 的 RL 分布式训练中，所有 rank 在开始更新前必须拥有完全相同的模型参数与 optimizer 状态。因为每个 rank 会在各自的数据/轨迹上独立计算梯度，DDP 只对梯度做 all-reduce（求和/平均）而不自动同步参数；若初始参数不同，则各 rank 计算的是不同参数点上的梯度，梯度平均不再对应单一参数点的有效下降方向，且后续 step 也无法保持参数同步，训练稳定性与收敛性都会显著变差。
+
+**核心操作：** 无论是随机初始化还是从检查点加载，都必须确保所有 rank 获得相同的参数。
+
+```python
+from humanoid.algo.iron_turbo.core import is_distributed, broadcast_parameters
+
+if is_distributed():
+    broadcast_parameters(actor_critic, discriminator)
+```
+
+**具体场景：**
+- **随机初始化后**：所有 rank 会自动获得相同的随机初始化参数（确保随机种子相同）。
+- **从检查点加载时**：如果只在 rank0 加载检查点，加载后必须广播给其他 rank。
+
+必要时在关键步骤后添加��步屏障：
+
+```python
+accelerator.wait_for_everyone()
+```
+
+下面是对这段的“更准确 + 更简洁 + 更像工程文档”的版本（顺便修正了一个关键点：`accelerator.backward(loss)` **不会**自动在各 rank 间平均 loss；它主要负责 AMP/梯度累积等反向传播封装）。
+

+
+### Step 5：每次更新中手动同步梯度（DDP-style RL 核心）
+
+IronTurbo 采用 **DDP-style**（不把模型包进 `DistributedDataParallel`），因此**反向传播不会自动触发梯度 all-reduce**。在 `AMPPPO.update()` 中使用如下固定模式：先本地反向得到梯度，再显式 all-reduce 并取全局平均，确保每个 rank 的 optimizer 看到一致的全局梯度。
+
+```python
+from humanoid.algo.iron_turbo.core import sync_gradients
+
+optimizer.zero_grad()
+
+# 1) 反向传播：accelerator.backward 主要封装 AMP / grad scaling / grad accumulation
+#    注意：这里不会自动做梯度同步
+if accelerator:
+    accelerator.backward(loss)
+else:
+    loss.backward()
+
+# 2) 添加梯度同步（关键）：对所有参数梯度做 all-reduce，并除以 world_size 得到全局平均梯度
+sync_gradients(actor_critic, discriminator)
+
+clip_grad_norm_(...)
+optimizer.step()
+```
+
+**关键说明**
+
+* `accelerator.backward(loss)`：本质是对 `loss.backward()` 的封装，负责 **AMP（GradScaler）**、**梯度累积** 等训练细节；**不等价于 DDP 的梯度同步**。
+* `sync_gradients(...)`：在模型未封装 DDP 的情况下，显式对每个参数的 `grad` 执行 `all_reduce(SUM)` 并除以 `world_size`，使每个 rank 的更新等价于“使用全局平均梯度更新”。
+
+**为什么采用这种方式**
+
+* 保留模型的自定义接口与调用方式（例如 `act / evaluate / get_actions_log_prob` 等），避免 DDP wrapper 对接口/访问路径的侵入。
+* 通过显式同步梯度，仍然获得与 DDP 类似的“多卡一致更新”语义（前提：各 rank 的初始化一致、每次 update 参与同步的参数集合一致）。
+
+## Step 6：同步训练过程中的统计量（用于日志指标）
+
+训练时很多指标是“每个 rank 各算一份”的。为了让日志反映**全局（所有 GPU 合并后）**的数值，需要在记录前做一次跨 rank 聚合。IronTurbo 提供两类常用同步方式：
+
+### 6.1 两种常用同步语义
+
+* **`sync_tensor(x, reduction="mean"/"sum")`：对“已经是聚合结果”的张量做跨 rank reduce**
+
+  * 适用：`x` 本身就是**每个 rank 已经算好的单个指标**（通常是标量 tensor），你希望把这些“每卡一份的指标”在 rank 维度做平均/求和。
+  * 例子：每卡各自算出的 `policy_loss`、`value_loss`、`kl`、`entropy`（已经是一个标量），想得到“多卡平均后的 loss”。
+
+* **`sync_tensor_mean(samples)`：对“样本集合”先取本地均值，再取全局均值**
+
+  * 适用：`samples` 是**每个 rank 上的一组样本值**（常见形状是 `[B]`、`[B, ...]`），你想得到“所有 rank 拼在一起后的整体均值”。
+  * 换句话说：每卡有一段数据（例如本卡 rollout 的 rewards、advantages、episode lengths），你想要汇总成一个全局均值用于日志。
+
+> “每个 rank 有一批样本”指的就是：**同一个指标在每个 GPU 上不是一个标量，而是一组样本（向量/矩阵），这些样本来自该 GPU 自己的 env/rollout/batch**。你要记录的是“所有 GPU 的样本合并后的整体均值”。
+
+### 6.2 示例：reward 的全局均值（每卡一组 reward 样本）
+
+```python
+from humanoid.algo.iron_turbo.distributed_utils import sync_tensor_mean
+
+# amp_reward 可能是 shape [B]（该 rank 上 B 个 reward 样本）
+amp_reward_mean = sync_tensor_mean(amp_reward, accelerator)
+```
+
+### 6.3 补充：指标字典同步（直接喂 logger）
+
+如果你有一个 `{name: scalar_tensor}` 的字典（losses/metrics），用 `sync_tensor_dict`：
+
+```python
+from humanoid.algo.iron_turbo.distributed_utils import sync_tensor_dict
+
+loss_dict_synced = sync_tensor_dict(loss_dict, accelerator, device)
+# 适合直接 logger.add_scalar(...)
+```
+
+## Step 7：全局归一化与统计一致性（Normalizer wrapper + 函数式标准化）
+
+在多卡训练中，凡是涉及 **normalize / 标准化 / running statistics** 的地方，都应确保使用的是**全局统计量**（跨所有 rank 聚合后的 mean/var），否则会出现“每卡各自归一化”，导致训练噪声增大、指标不可比甚至收敛变差。IronTurbo 提供两类接口，覆盖两种典型场景：
+
+### 7.1 状态型归一化：RunningMean / Normalizer 的跨 rank 同步（最小侵入 wrapper）
+
+用于 **带状态的归一化器**（例如 `NormalizerTorch` / `RunningMeanStdTorch`），其核心是维护 `mean/var/count` 并随训练持续更新。迁移时只需在构造后包一层 wrapper：
+
+```python
+from humanoid.algo.iron_turbo.core import SyncNormalizerTorch
+
+amp_normalizer = SyncNormalizerTorch(amp_normalizer)
+```
+
+**语义**
+
+* 单卡：行为不变（直接走本地 update）。
+* 多卡：在 `update()` 内部对本 rank 的统计量做 `all_reduce` 聚合，再更新到底层 normalizer，从而保证所有 rank 的 running stats 始终一致。
+
+**优势**
+
+* 无需修改原有 normalizer 实现；wrapper 以“组合”的方式注入分布式同步。
+* 保存/加载仍可直接使用 `state_dict()`（wrapper 将底层 normalizer 注册为子模块，ckpt 兼容）。
+
+---
+
+### 7.2 无状态归一化：对某个张量做一次性全局标准化（例如 GAE/advantages）
+
+用于 **不需要维护长期状态**、只对当前张量做一次标准化的场景，例如 rollout storage 里对 `advantages` 做 normalize。建议使用函数式原语：
+
+```python
+from humanoid.algo.iron_turbo.core import standardize_tensor
+
+advantages = standardize_tensor(advantages, accelerator)
+```
+
+这样 advantages 的均值/方差来自**全局统计**（所有 rank 合并后的 mean/std），而非每卡独立统计。
+
+---
+
+### 如何选接口（经验法则）
+
+* **需要跨 step/epoch 维护 mean/var/count（running stats）** → 用 `Sync*Normalizer*` wrapper（状态型）
+* **只对当前 batch/tensor 做一次 normalize（无长期状态）** → 用 `standardize_tensor`（函数式）
+
+
+## Step 9：对齐“必须一致”的随机采样（例如专家数据预加载）
+
+当某些采样必须在所有 rank 间保持一致时（否则会导致 buffer 或统计不一致），可按照 `motion_loader.py` 的做法：
+
+1. 仅在 rank0 进行采样；
+2. 将结果广播至其他 rank 进行覆盖。
+
+```python
+from humanoid.algo.iron_turbo.core import is_distributed
+from humanoid.algo.iron_turbo.distributed_utils import broadcast_overwrite_from_src
+
+if is_distributed():
+    traj_idxs, times = broadcast_overwrite_from_src(
+        [traj_idxs, times], src=0, device=device, return_numpy=True
+    )
+```
+
+## 迁移检查清单（最小闭环）
+
+* **1) 启动方式**：用 `python -m torch.distributed.run`（或 `torchrun`）启动单机多卡，确保每卡一个进程、`world_size>1` 生效
+* **2) 设备映射 + per-rank seed（train 入口）**：用 `local_rank` 绑定 `sim_device/rl_device = cuda:{local_rank}`；训练前设置 per-rank seed（如 `base_seed + rank` / `get_distributed_seed`）
+* **3) 训练开始前参数对齐**：初始化/加载后用 `broadcast_parameters(...)` 保证各 rank 参数完全一致（必要时 `wait_for_everyone()`）
+* **4) 梯度同步（DDP-style 核心）**：每次 `backward` 后、`optimizer.step()` 前调用 `sync_gradients(...)` 做全局平均梯度
+* **5) 全局归一化（Normalizer / AMP normalize）**：对 running stats 类 normalizer 用 `Sync*` wrapper（如 `SyncNormalizerTorch`）；对一次性张量标准化用 `standardize_tensor(...)`

+* **7) 全局一致的预加载/采样项**：对必须跨 rank 完全一致的数据（如 expert preload 索引/时间戳/预加载 buffer 初始化）用 `broadcast_overwrite_from_src(...)` 以 rank0 为准广播覆盖

+# IronTurbo Primitives（分布式训练原语）
+
+> 说明：IronTurbo 的核心实现位于 `humanoid.algo.iron_turbo`（历史代码/注释中可能仍出现 iron_turbo）。本文档介绍 IronTurbo 对分布式训练做的“最小侵入式”抽象：不强制 DDP 包装模型，但提供一组可组合的同步/聚合/归一化原语，帮助把单卡 RL 训练迁移为单机多卡 DDP-style 训练。
+
+## 设计分层
+
+IronTurbo 原语分两层：
+
+- **L0：通用分布式原语（底层通信/聚合）**
+  - 解决“跨 rank 同步/广播/聚合”的通用问题
+  - 典型用途：参数对齐、梯度 all-reduce、日志指标聚合、强制对齐采样结果、快速一致性检查
+
+- **L1：RL 训练中间层原语（全局统计/全局归一化）**
+  - 解决 DDP-style RL 中“必须全局一致”的统计与归一化问题
+  - 典型用途：advantages 全局标准化、running mean/std 的跨 rank 同步更新
+
+## 0. 基础判断：当前是否分布式
+
+> 这三者是所有分布式逻辑的 guard / 分支依据。
+
+### `get_world_size(default: int = 1) -> int`（`core.py`）
+稳健获取 world size（即使未初始化 process group 也不会报错）。
+
+- 已初始化分布式：返回 `dist.get_world_size()`
+- 否则：返回 `default`（默认 1）
+
+### `is_distributed() -> bool`（`core.py`）
+判断是否处于分布式模式：`get_world_size() > 1`。
+
+> 建议：所有 “distributed-only” API 调用前先 `if is_distributed(): ...`
+
+### `is_main_process(accelerator: Optional[Accelerator] = None) -> bool`（`core.py`）
+判断当前是否主进程（rank 0）。
+
+- 传入 `accelerator`：使用 `accelerator.is_main_process`
+- 不传：fallback 到 `torch.distributed`（未初始化则视为单进程主进程）
+
+用途：只在主进程写日志 / 保存 ckpt / 上传产物 / 打印关键配置。
+
+## 1) L0：参数对齐与梯度同步（`torch.distributed`）
+
+### `broadcast_parameters(*modules: nn.Module, src: int = 0) -> None`（`core.py`）
+将若干个模块的参数从 `src` 广播到所有 rank。
+
+**使用场景**
+- rank0 负责 load checkpoint / 初始化权重后：用广播保证所有 rank 参数完全一致
+- 训练开始前做一次参数对齐（DDP-style 必备）
+
+**行为**
+- 非分布式：no-op
+- 分布式：逐参数 `dist.broadcast(param.data, src)`
+
+**示例**
+```python
+if is_distributed():
+    broadcast_parameters(actor_critic, discriminator)
+````
+
+### `sync_gradients(*modules: nn.Module) -> None`（`core.py`）
+
+手动同步梯度（DDP-style RL 核心）：对每个 module 的 `param.grad` 做全局平均。
+
+**做了什么**
+
+1. flatten 当前 module 下所有非空 grad
+2. `dist.all_reduce(SUM)`
+3. 除以 `world_size` 得到平均梯度
+4. 再拆回写回每个参数的 grad
+
+**调用位置**
+
+* `backward()` 之后、`optimizer.step()` 之前
+
+**重要注意**
+
+* 如果模型已经被 `DistributedDataParallel` 包装（或其它框架自动做了梯度同步），**不要再调用**该函数，否则会重复 all-reduce。
+* 该实现依赖各 rank 对参数的遍历顺序一致（通常对同一份模型代码成立）。
+
+**示例**
+
+```python
+optimizer.zero_grad()
+
+if accelerator:
+    accelerator.backward(loss)   # AMP / grad scaling / grad accumulation 封装（不负责梯度 all-reduce）
+else:
+    loss.backward()
+
+sync_gradients(actor_critic, discriminator)
+
+clip_grad_norm_(...)
+optimizer.step()
+```
+
+## 2) L0：日志指标聚合（基于 `accelerator.reduce`）
+
+> 目标：业务侧直接同步 logging 指标，避免手写 all_reduce；统一采用 `Accelerate` 的 reduce 接口。
+
+### `sync_tensor(tensor: Tensor, accelerator=None, reduction: str = "mean") -> Tensor`（`distributed_utils.py`）
+
+对任意 shape 的 tensor 做跨 rank reduce，输出 shape 不变。
+
+* 非分布式 / `accelerator is None`：原样返回
+* 分布式：`accelerator.reduce(tensor, reduction="mean"/"sum")`
+
+适用：已经是“每 rank 一个指标”的情况，例如 `policy_loss`、`entropy`、`kl`（标量或小张量）。
+
+### `sync_tensor_mean(tensor: Tensor, accelerator=None) -> Tensor`（`distributed_utils.py`）
+
+先算本 rank 的 `local_mean`，再对这个标量做跨 rank mean reduce。
+
+适用：每个 rank 上拿到一组样本值（如 reward 向量、episode length 向量），想记录“全局均值”。
+
+**关键前提（容易踩坑）**
+
+* 该实现等价于 `mean( mean(rank_i_samples) )`，只有在 **各 rank 样本数相同** 时才等价于严格全局均值。
+* 若各 rank 样本数可能不同，需要严格全局均值请使用 `reduce_mean()`（见下文 L1）。
+
+**示例**
+
+```python
+amp_reward_mean = sync_tensor_mean(amp_reward, accelerator)
+```
+
+### `sync_tensor_dict(tensor_dict: Dict[str, Tensor], accelerator=None, device=None) -> Dict[str, float]`（`distributed_utils.py`）
+
+同步一个指标字典，并把值转成 Python float，适合直接喂给 logger。
+
+* 非分布式：`v.item()` 后返回
+* 分布式：对每个 key 的 value 做 `accelerator.reduce(..., "mean")`
+* 关键细节：按 `sorted(keys)` 遍历，保证各 rank collective 调用顺序一致
+
+**示例**
+
+```python
+loss_dict_synced = sync_tensor_dict(loss_dict, accelerator, device)
+```
+
+
+## 3) L1：全局统计与全局标准化（Advantage / 统计量）
+
+### `reduce_mean(tensor: Tensor, accelerator=None) -> Tensor`（`core.py`）
+
+计算严格的全局均值（sum + count 方式），不依赖各 rank 样本数相同。
+
+* 单卡：返回 `tensor.mean()`
+* 多卡：计算本地 sum 与 count，`accelerator.reduce(..., "sum")` 后做 `global_sum / global_count`
+
+适用：当各 rank 的样本数可能不同，但你仍希望统计是严格全局均值（例如不等长 episode 统计、过滤后的样本等）。
+
+
+### `standardize_tensor(tensor: Tensor, accelerator=None, eps=1e-8) -> Tensor`（`core.py`）
+
+对 tensor 做**全局**标准化（zero mean / unit std）。
+
+* 单卡：本地 mean/std
+* 多卡：先用 reduce 得到全局 mean，再聚合平方差得到全局方差（并行统计方式）
+
+典型用途：advantages（GAE）全局 normalize，避免每卡各自 normalize 引入额外噪声。
+
+
+## 4) L1：Running Stats 同步封装（最小侵入 wrapper）
+
+> 思路：底层 normalizer 维持本地实现；wrapper 只在 `update()` 处注入分布式同步。
+> wrapper 把 base 注册为 submodule，因此 `state_dict()` 保存/加载天然兼容。
+
+### `SyncRunningMeanStd(base: nn.Module)`（`core.py`）
+
+对 `RunningMeanStdTorch` 的 drop-in wrapper：
+
+* 单卡：`update()` 直接调用 base.update
+* 多卡：`update()` 内计算本地统计量，`dist.all_reduce(SUM)` 聚合成全局 mean/var/count，再调用 `base.update_from_moments(...)`
+
+特点：
+
+* 透传 `mean/var/count`
+* wrapper 可直接 `state_dict()/load_state_dict()`
+
+### `SyncNormalizerTorch(base: nn.Module)`（`core.py`）
+
+对 `NormalizerTorch` 的轻量 wrapper（语义同上）：
+
+* 单卡：直接 `base.update(arr)`
+* 多卡：基于当前 mean 做 shift，再聚合 sum/sumsq 得到全局 mean/var，最后 `update_from_moments(...)`
+
+**用法**
+
+```python
+amp_normalizer = NormalizerTorch(...)
+amp_normalizer = SyncNormalizerTorch(amp_normalizer)
+```
+
+## 5) L0：强制对齐“必须一致”的采样/预加载（任意 dtype/shape 广播覆盖）
+
+### `broadcast_overwrite_from_src(xs, src=0, device="cuda", return_numpy=False) -> Tuple[Tensor, ...]`（`distributed_utils.py`）
+
+用于“由 src rank 决定数据，其它 rank 强制覆盖对齐”。支持 numpy/torch 输入，支持任意 shape/dtype（在支持列表内）。
+
+**语义**
+
+* 必须在所有 rank 调用（collective）
+* 只有 `src` rank 的输入是权威数据，其它 rank 的输入会被忽略
+* 对每个 item：先广播 dtype/ndim/shape，再广播 payload
+
+**前置条件**
+
+* 分布式已初始化且 world_size>1（否则会抛异常；调用前请 `if is_distributed(): ...`）
+
+**典型用途**
+
+* expert preload 的索引/时间戳必须全局一致：rank0 采样后广播覆盖到其它 rank
+
+## 6) L0：分布式 seed
+
+### `get_distributed_seed(base_seed: int, accelerator=None) -> int`（`distributed_utils.py`）
+
+返回 `base_seed + accelerator.process_index`（单卡/无 accelerator 则返回 `base_seed`）。
+
+用途：每 rank seed 偏移，保证采样/探索具有多样性。

+    def assemble_obs_buffer(
+        self,
+        obs_proprio_history,
+        obs_extero_history,
+        privileged_obs_history,
+        privileged_obs_extero_history,
+        random_coef,
+    ):
+        """Assemble the observations and privileged observations into a single tensor."""
+        # ---- obs_buf (actor) ----
+        delay = random.randint(0, self.cfg.env.itof_delay_std) + self.cfg.env.itof_delay
+        length_obs_buf = obs_proprio_history.maxlen - self.cfg.env.itof_delay - self.cfg.env.itof_delay_std
+        T = length_obs_buf
+        K = self.cfg.env.num_frames_extero
+        B = obs_proprio_history[0].shape[0]
+        Dp = obs_proprio_history[0].shape[1]
+        De = obs_extero_history[0].shape[1]
+        total_obs_dim = T * Dp + K * De
+        obs_buf = obs_proprio_history[0].new_empty(B, total_obs_dim)
+
+        col = 0
+        for i in range(T):
+            p = obs_proprio_history[i + delay]  # [B, Dp]
+            obs_buf[:, col : col + Dp] = p
+            col += Dp
+            if T - i <= K:
+                e = obs_extero_history[i]  # [B, De]
+                if not torch.any(e):
+                    n = len(obs_extero_history)
+                    for j in range(i + 1, n):
+                        cand = obs_extero_history[j]
+                        if torch.any(cand):
+                            e = cand
+                            break
+                obs_buf[:, col : col + De] = e
+                col += De
+
+        # ---- privileged_obs_buf (critic) ----
+        c = self.cfg.env.c_frame_stack
+        B = privileged_obs_history[0].shape[0]
+        D_hist = privileged_obs_history[0].shape[1]
+        D_ext = privileged_obs_extero_history[0].shape[1]
+
+        total_priv_dim = c * (D_hist + D_ext)
+        buf = privileged_obs_history[0].new_empty(B, total_priv_dim)
+
+        col = 0
+        for i in range(c):
+            p = privileged_obs_history[i]  # [B, D_hist]
+            e = privileged_obs_extero_history[-c + i]  # [B, D_ext]
+            buf[:, col : col + D_hist] = p
+            col += D_hist
+            buf[:, col : col + D_ext] = e
+            col += D_ext
+
+        privileged_obs_buf = torch.cat((buf, random_coef), dim=1)
+
+        return obs_buf, privileged_obs_buf

+    def collect_data(self):
+        if self.common_step_counter % 10 == 0:
+            logging.info(
+                f"[COLLECT] Step {self.common_step_counter}, file_buf: {len(self.file_buf)}/{self.cfg.record.file_total_data}"
+            )
+
+        added_to_file_buf = 0
+        reset_envs = 0  # Number of environments that just reset
+        skipped_no_occlusion_envs = 0
+
+        for i in range(self.num_envs):
+            if self.reset_buf_record[i] > 0:
+                self.sim_buf[i].clear()
+                self.reset_buf_record[i] = 0
+                reset_envs += 1
+                continue
+
+            if self.cfg.record.record_occlusion:
+                if not self.occ_env_buffer[i]:
+                    # logging.info(f"Env {i}: no occlusion, skipping data collection")
+                    skipped_no_occlusion_envs += 1
+                    continue
+
+            # construct sample: for tensors, slice by environment, for category, directly use
+            sample = {
+                key: (value if key == "category" else value[i].cpu().numpy().copy())
+                for key, value in self.cur_states.items()
+            }
+
+            self.sim_buf[i].append(sample)
+
+            # if the data of this environment is not long enough, skip
+            if len(self.sim_buf[i]) < self.cfg.record.record_deque_buffer_size:
+                continue
+
+            # take the first data_seq_len steps, and form a sequence
+            seq = [self.sim_buf[i].popleft() for _ in range(self.cfg.record.data_seq_len)]
+            self.file_buf.append(seq)
+            added_to_file_buf += 1
+
+        if self.common_step_counter % 10 == 0:
+            logging.info(
+                f"[COLLECT] Added to file_buf: {added_to_file_buf}, Reset envs: {reset_envs}, Skipped(no occlusion): {skipped_no_occlusion_envs}"
+            )
+            # logging.info(f"[COLLECT] Added to file_buf: {added_to_file_buf}, Reset envs: {reset_envs}")
